## Getting Started

### Quick Start (NativeVibe Local Dev)

1. Open your project on [NativeVibe](https://nativevibe.dev) and go to the **Local Dev** tab
2. Download and run `nv-setup.sh` — it installs a pinned Flutter SDK, clones your repo, and sets up sync
3. Run `flutter run` to launch on a simulator/device with automatic hot reload from cloud changes

### Manual Setup

1. Install Flutter (3.24.x recommended): https://flutter.dev/docs/get-started/install
2. Clone this repository
3. Run `flutter pub get` to install dependencies
4. Run `flutter run` to start the app
